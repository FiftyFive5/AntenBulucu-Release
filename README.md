# Anten Bulucu - Indir

MD (link) antenlerinin (Radwin, InfiNet, Repeatit vb.) IP adresini bulan,
PC'ye otomatik uygun IP tanimlayip web arayuzunu acan Windows uygulamasi.

## Kurulum

1. [Npcap](https://npcap.com) kurun ("WinPcap API-compatible mode" isaretli).
2. Sag taraftaki **Releases** bolumunden en son `AntenBulucu.exe` dosyasini indirin.
3. Calistirin (yonetici yetkisi ister, kendisi ister).

Baska bir kurulum gerekmez; Python kurulu olmasina gerek yoktur.
